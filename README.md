**Ghibli Sky Weather** is a web-based weather application inspired by Studio Ghibli aesthetics. It combines real-time weather data with interactive visuals and ambient sound effects to provide an immersive and relaxing user experience.

---

## Features

- **Real-Time Weather Data**  
  - Current temperature, weather condition, humidity, wind speed  
  - Sunrise and sunset times  
  - 5-day weather forecast

- **Dynamic Themes & Animations**  
  - Day mode: bright sky with moving clouds and flying birds  
  - Night mode: dark sky with stars, glowing moon, shooting stars, and a flying airplane  
  - Parallax cloud animations influenced by wind speed

- **Weather Effects**  
  - Rain: animated raindrops  
  - Snow: falling snowflakes  
  - Shooting stars at night

- **Weather-Based Background Music**  
  - Calm tunes for clear weather  
  - Rain sounds for rainy conditions  
  - Snowy ambiance for snow  
  - Toggle to turn sound on or off

- **Manual Day/Night Toggle**  
  Users can switch between day and night themes manually.

1. Open `index.html` in a web browser.  
2. Enter the name of a city in the search box and click **Search**.  
3. View the current weather, extra details, and the 5-day forecast.
